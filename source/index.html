X-платформенная среда разработки

  • ModBus TCP/IP 1

    Открытый коммуникационный протокол, основанный на архитектуре ведущий — ведомый

  • OPC UA

    Новейший протокол, основанный не на технологии Microsoft COM, что предоставляет кроссплатформенную совместимость

  • OPC DA

    Описывает набор функций обмена данными в реальном времени с ПЛК и другими устройствами

  • C# Dataprovider

    Позволяет работать с данными и сторонними источниками не входящих в контур АСУТП

  • ONNX

    Протокол поддержки аналитических моделей на основе глубокого нейросетевого обучения

  • MSSQL

    Позволяет работать с реляционной системой управления базами данных MSSQL, получать и изменять данные в таблицах

  • Apache Kafka

    Для интеграции SuperSCADA с ПО Верхнего Уровня MES\CRM\BI\ERP

  • MQTT

    Сетевой протокол ориентированный на обмен сообщениями между устройствами по принципу «издатель — подписчик»
